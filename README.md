# Euphemia Database Autonomous Test

This repository contains SQL scripts and related files for performing autonomous tests in the **Euphemia Database**.

## Contents

- `inputs.sql` – Main SQL script used for autonomous testing.
- Other utility scripts (may be added in the future):
  - `createUser.sql` – Script to create a new user.
  - `deleteUser.sql` – Script to delete a user.
  - etc.
